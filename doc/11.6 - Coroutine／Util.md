# Coroutine／Util

##方法列表
### 1. getuid
```
int \Swoole\Coroutine::getuid()
```
获取当前协程的唯一`ID`
**返回值**：

* 成功时返回当前协程`ID（int）`
* 如果当前不在协程环境中，则返回`-1`

```php
echo \Swoole\Coroutine::getuid();
```

协程设置
---
可使用`Coroutine::set`设置协程相关选项。
```php
Swoole\Coroutine::set(array(
	'max_coroutine' => 4096,
));
```

#### max_coroutine
设置最大协程数，超过限制后底层将无法创建新的协程。

#### stack_size
设置单个协程初始栈的内存尺寸，默认为`8192`
