# Coroutine\MySQL

启用协程MySQL客户端
----
* 需要在编译swoole时增加--enable-coroutine来开启此功能。

使用示例
---
```php
$swoole_mysql = new Swoole\Coroutine\MySQL();
$swoole_mysql->connect([
	'host' => '127.0.0.1',
	'port' => 3306,
	'user' => 'user',
	'password' => 'pass',
	'database' => 'test',
]);
$res = $swoole_mysql->query('select sleep(1)');
```
